# Icons

9ui uses [`lucide-react`](https://lucide.dev/) for icons.

For a more extensive collection of icons, consider using [`Monicon`](https://monicon-docs.vercel.app/), which offers over 200,000 icons from various popular libraries.

If you are looking for animated icons, the [`pqoqubbw/icons`](https://icons.pqoqubbw.dev/) library is a great choice.

```md title="based-ui.mdc"
# Setup based-ui

## 1. Initialize shadcn/ui in your project

Choose your package manager and run the init command:

### pnpm

pnpm dlx shadcn@2.3.0 init

If you don't have tailwind installed, install 3.4.15

pnpm add tailwindcss@3.4.15 postcss autoprefixer

Follow the prompts. Choose between a Next.js project or a Monorepo.

## 2. Install Base UI

### pnpm

pnpm add @base-ui-components/react

## 3. Add Root class to your app layout

Update your layout file (e.g., `layout.tsx` or `layout.jsx`):


// layout.tsx or layout.jsx
<body>
  <div className="isolate">{children}</div>
</body>

## 4. Add 9ui theme to your project

### pnpm
pnpm dlx shadcn add "https://ui.fldr.zip/r/theme.json"

## 5. That's it! You can now install the components.
### pnpm

npx shadcn@latestadd https://ui.fldr.zip/r/button.json

```
